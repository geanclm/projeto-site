CORS - Cross-Origin Resource Sharing (Compartilhamento de recursos com 
origens diferentes) é um mecanismo que usa cabeçalhos adicionais HTTP para 
informar a um navegador que permita que um aplicativo Web seja executado em 
uma origem (domínio) com permissão para acessar recursos selecionados de um 
servidor em uma origem distinta. Um aplicativo Web executa uma requisição 
cross-origin HTTP ao solicitar um recurso que tenha uma origem diferente 
(domínio, protocolo e porta) da sua própria origem.
